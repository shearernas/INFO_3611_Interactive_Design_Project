Topic 1: What is MySQL

MySQL is a completely free and open-source implementation of the SQL querying language.

It allows you to create, modify, and manipulate data in a database.

It is classed as a "relational database management software", which means it used for querying data from relational tables (no worries if you don't know what this is, more on this in a bit).

MySQL is used by millions of websites on the Internet to store and retrieve data for critical website functions.

Many companies around the world trust MySQL for storing confidential and important user and product data because it is secure, reliable, and quick.
