Topic 4: Installing MySQL on macOS

Once it has finished downloading, double click on the file to mount the installation package to your computer. It should be named "mysql-9.2.0-macos15-x86_64.dmg" or "mysql-9.2.0-macos15-arm64.dmg".

After you mount the package, a new window should appear displaying the contents. Right click on the single file with the open box icon to begin the installation process.
